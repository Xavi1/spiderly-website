Mandatory Prerequisites

  • SQL Server (Free Developer Edition)
  • Node.js
  • Angular CLI: npm install -g @angular/cli
  • Code Editors (Recommended):
    • Backend: Visual Studio (Community Edition)
    • Frontend: Visual Studio Code
  • .NET 9.0 (if you have installed Visual Studio, you can skip this step)
  • EF Core CLI: dotnet tool install -g dotnet-ef

Optional Recommended Tools

  • SQL Server Management Studio
  • Visual Studio Code Extension: Angular Language Service
